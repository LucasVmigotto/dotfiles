# dotfiles

Script that allow a fast terminal configuration and programs/tools installation such as:

* [VSCode](https://code.visualstudio.com/) (+ [Extensions](https://gist.github.com/LucasVmigotto/47904ee4f36b70c9d84a30d29d996aed))
* [`git`](https://git-scm.com/) (+ [`git-lfs`](https://git-lfs.github.com/))
* [Docker](https://docs.docker.com/engine/install/ubuntu/) (+ [Docker Compose](https://docs.docker.com/compose/install/))
* [`asdf`](https://asdf-vm.com/#/core-manage-asdf-vm)
* [`yarn`](https://classic.yarnpkg.com/en/docs/install/#debian-stable)
* [VLC](https://www.videolan.org/)
* [Transmission](https://transmissionbt.com/)
* [`tree`](https://linux.die.net/man/1/tree)
* [OpenVPN](https://openvpn.net/)
* xclip

## Usage

1. Clone the repository

    ```bash

    git clone https://github.com/LuasVmigotto/dotfiles.git ~/dotfiles

    ```

2. Execute the `install.sh`

    ```bash

    ./$HOME/dotfiles/install.sh

    ```
